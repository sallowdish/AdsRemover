AdsRemover

Tool to remove and block annoying iframe Ads which is non-related to the topic of the page.

Usage

  1. Click the 'Bookmark ME' button
  2. Copy the code display on page
  3. Create a new Bookmark in your toolbar
  4. Paste the code to the 'location' feild of the new bookmark
  5. Click the bookmark whenever ads need to be removed
